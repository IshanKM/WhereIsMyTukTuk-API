# WhereIsMyTukTuk
WhereIsMyTukTuk is a real-time vehicle tracking system designed to monitor and manage TukTuk locations efficiently. The system is built using Node.js, Express.js, and MongoDB, following a scalable REST API architecture.

It supports multiple user roles including Admin, Operator, and Police, with secure JWT-based authentication and role-based access control. The platform allows real-time GPS location updates, historical tracking, and district-based filtering.

A custom simulator is included to generate live location data, making it suitable for testing real-time tracking scenarios without physical devices.

Key Features:
- Real-time TukTuk location tracking
- Role-based authentication (Admin, Operator, Police)
- RESTful API with Swagger documentation
- MongoDB-based scalable data storage
- GPS simulator for testing
- Dockerized deployment with Docker (DigitalOcean ready)
